### Выбор папки или файлов

Компонент можно настроить, чтобы была возможность выбирать только файлы, только папки, файлы или папки.

<!-- example(file-upload-allowed) -->

По умолчанию выбранная папка будет показана в списке в виде одной записи, а для множественного загрузчика можно настроить, чтобы после выбора каталога в список попали все вложенные файлы в виде отдельных элементов.

### Неактивное состояние

В неактивном состоянии компонент не пропадает в фокус, не работает механика перетаскивания.
