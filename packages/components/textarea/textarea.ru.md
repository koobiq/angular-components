`KbqTextarea` — компонент для многострочного ввода текста.

<!-- example(textarea-overview) -->

### Автоматическое изменение размера

При включении атрибута `canGrow` текстовое поле будет автоматически изменять высоту в зависимости от содержимого.

<!-- example(textarea-can-grow) -->

### Максимальное количество строк

Атрибут `maxRows` ограничивает максимальное количество строк текстового поля, при достижении лимита появляется прокрутка.

<!-- example(textarea-max-rows) -->

### Неактивное состояние

Поле недоступно для редактирования и взаимодействия. Настраивается при помощи атрибута `disabled`.

<!-- example(textarea-disabled) -->

### Состояние ошибки

Отображение поля в состоянии ошибки для визуальной индикации невалидных данных.

<!-- example(textarea-error-state) -->
