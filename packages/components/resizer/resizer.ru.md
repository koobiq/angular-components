Директива для изменения размера элемента в заданном направлении при помощи указателя мыши.

<!-- example(resizer-overview) -->

### Направление изменения размера

Устанавливается атрибутом `kbqResizer` в формате массива `[x, y]`, где:

- `x` - горизонтальное направление: `-1` (влево), `0` (отключено), `1` (вправо).
- `y` - вертикальное направление: `-1` (вверх), `0` (отключено), `1` (вниз).

```html
<div kbqResizable>
    <i [kbqResizer]="[1, 0]"></i>
</div>
```
