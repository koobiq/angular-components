Компонент `kbq-flag` показывает небольшой флаг страны. Сам флаг (SVG или изображение) передаётся через проекцию контента, а компонент отвечает только за оформление: форму, размер, тень и доступность.

Изображения флагов удобно брать из пакета [country-flag-icons](https://www.npmjs.com/package/country-flag-icons):

- Полное покрытие ISO 3166-1. Все существующие страны обозначаются двухбуквенным кодом (RU, DE, FR).
- Флаги перерисованы под мелкий размер: детали упрощены, поэтому флаги остаются узнаваемыми даже в размере иконки.
- Активная поддержка и лицензия MIT.

## Соотношение сторон

По умолчанию флаги имеют соотношение 3:2. Это основной формат — используйте его в большинстве сценариев: инлайн с текстом, списки, селекты.

Квадратные флаги (`shape="square"`) нужны там, где формат 3:2 не подходит: селекторы аватаров, компактные кружки, плотные сетки. Для них передавайте квадратный вариант изображения (подпакет `1x1`). Большинство версий 1:1 получены обрезкой 3:2 по центру; для флагов, теряющих смысл при таком кадрировании, в пакете есть отдельные квадратные варианты (например, EU и KR).

## Внутренняя тень: отделение от фона

Флагу добавляется тонкая внутренняя тень (inset-обводка) по краю. Без неё флаг сливается с фоном: много белого — со светлым фоном страницы, чёрный или тёмный — с фоном в тёмной теме.

Цвет тени зависит от темы: в светлой теме тень тёмная, в тёмной — светлеет. Тень включена по умолчанию (`shadow="inset"`); отключить её можно через `shadow="none"`.

## Флаг — это не язык

Флаг обозначает страну или регион, а не язык. Не используйте флаг для выбора языка интерфейса: один язык бывает в нескольких странах (испанский — не только Испания), а в одной стране бывает несколько языков.

## Если нужного флага нет

Сначала проверяйте наличие через `hasFlag(code)` и не показывайте пустоту:

```ts
import { hasFlag } from 'country-flag-icons';

hasFlag('RU'); // true
hasFlag('ZZ'); // false
```

Для неизвестного или некорректного кода показывайте нейтральную заглушку — добавьте класс `kbq-flag_empty` и не проецируйте изображение.

<!-- example(flag-fallback) -->

Если нужен флаг не-страны (организация, историческая или спорная территория, субъект федерации) — не добавляйте его в пакет country-flag-icons, а спроецируйте собственное изображение в тот же `kbq-flag`, чтобы способ использования не менялся. Для наднациональных флагов (кроме ЕС) создавайте в проекте собственные изображения. Для показа в крупном размере используйте флаги в оригинальной детализации из стороннего пакета — в country-flag-icons они упрощены под мелкий размер.

## Доступность

У флага всегда должна быть текстовая альтернатива. Если флаг несёт смысл и рядом нет текста — передавайте подпись через `label`: компонент получит `role="img"` и доступное имя. Если рядом уже есть текст (инлайн, опция, блок) — флаг декоративный, помечайте его `decorative`, чтобы он скрывался от экранного диктора и не дублировал подпись.
