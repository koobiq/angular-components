База шрифта иконок .kbq { transform: rotate(0.001deg) } (специфичность 0-1-0) конкурирует с хелперами иконки .kbq-icon-rotate_* / .kbq-icon-flip-* (тоже 0-1-0). В режиме adoptedStyleSheets без слоя база сортируется в каскаде последней и побеждает — повороты и флипы пропадают. Обёртка в @layer kbq-icons возвращает приоритет переопределениям компонентов.

1. Повороты и флипы — наглядный баг

Хелперы из icon.scss . В режиме «БАГ» иконки распрямляются в 0°; в «ФИКС» и «эталон» — повёрнуты.

без класса (0°) rotate_90 rotate_180 rotate_270 flip-h flip-v flip-vh

2. Input с password toggle — смещение иконки

Иконка-глаз ( kbq-password-toggle , это kbq-icon-button ) в суффиксе поля. В режиме «БАГ» база .kbq ломает выравнивание — глаз уезжает по вертикали; в «ФИКС»/«эталон» — на месте.
