# SSR dev application

Development application for testing SSR features of Koobiq Angular Components.

Based on [Angular SSR example](https://github.com/angular/angular/tree/18.2.14/adev/src/content/examples/ssr) from the [official documentation](https://v18.angular.dev/guide/ssr).

## Development

Make sure you have the [correct version](.nvmrc) of Node.js installed (we recommend using [nvm](https://github.com/nvm-sh/nvm)).

```bash
# Setup Node.js
nvm use

# Install dependencies
yarn install

# Run application
yarn run dev:ssr
```
