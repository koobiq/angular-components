Toast inside an emulated MFE (Shadow DOM)

This app root uses ViewEncapsulation.ShadowDom with the kbq-light theme scoped to the shadow root. Open with ?container=default to see the bug (the toast escapes to document.body and loses its theme tokens) and with ?container=shadow to see the fix ( provideKbqShadowDomOverlay keeps the toast inside the shadow root).
