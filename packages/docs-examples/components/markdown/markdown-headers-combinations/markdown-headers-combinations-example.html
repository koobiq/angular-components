## Описание работы ### Длина сообщения #### Виды тостов Рекомендованная длина сообщения — 144 символа, заголовка — 80 символов. Длинный текст не обрезается, тост увеличивается по высоте. ### Появление сообщений Тост появляется в правом верхнем углу. Несколько сообщений образуют стопку на экране. Новые сообщения появляются внизу списка. Если тостов много, то новые уходят даже за нижнюю границу видимой области. Чаще всего тост открывается в качестве реакции на команду пользователя, но система может показывать уведомления и самостоятельно.
